# Analytics

The **Analytics & Performance** tab shows usage metrics for each prompt.

## Overview Cards

- **Total Prompts**: Active templates in the project
- **Usage (7 days)**: Total API calls fetching prompts
- **Cost (7 days)**: Associated LLM costs from agents using these prompts

## Version Analytics

Each version tracks:

- Number of times fetched
- Success/error rates
- Latency distribution
- Token usage

## Traces

View individual traces that used a specific prompt version:

1. Open the prompt detail page
2. Select a version
3. Click the **Traces** tab
4. Browse traces with full context
